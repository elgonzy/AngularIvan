Aquesta és una unitat d'un component senzill d'estil jumbotron per cridar l'atenció addicional al contingut o la informació destacada.

  • Crear un projecte senzill a Angular
  • Entendre l'ús de rutes a Angular
  • Usar el component Router per a la càrrega dinàmica de pàgines
  • Usar el patró d'injecció de dependències per a l'ús de serveis i captura de paràmetres
  • Crear serveis de generació de dades síncrons
  • Crear serveis asíncrons basats en APIs REST
  • Consumir de serveis de forma asíncrona
  • Presentar dades dels components a les vistes
Més informació

Passes per a construir el nostre projecte @Angular

  • Creació del projecte
    1. ng new 04_NavApp
  • Instal·lació de Bootstrap
    1. npm install bootstrap jquery @popperjs/core --save
    2. Incloure en la secció architect.build d'angular.json les referències al CSS i json
      Això equival a incloure a l'index.html el CSS i JS
  • Creació dels components
    1. ng generate component components/home
    2. ng generate component components/about
    3. ng generate component components/shared/navbar
    4. Aquests components són generats de manera automàtica a l'array de declarations en app.module.ts
  • Enrutament
    1. Les rutes es trobaran definides a l'arxiu app.route.ts
    2. Aquest nou fitxer creat s'ha d'importar a app.module.ts
    3. S'ha d'afegir a l'array imports RouterModule.forRoot(ROUTES,{useHash:true})
    4. A la pàgina app.component.html haurem de posar el selector router-outlet
  • Servei de dades en memòria
    1. Crearem el servei amb la comanda ng g s services/empleats
    2. Recordem que hem de posar manualment el servei EmpleatsService dins la seccció providers al fitxer app.module.ts
    3. Complement el servei declarant un array d'empleats
    4. Creem dos mètodes dins el servei
  • Creació del component d'empleats
    1. La comanda ng g c components/empleats
    2. Recordem que hem d'actualitzar el fitxer de rutes
    3. Recordem que hem de modificar la vista del menú
    4. Implementar el controlador d'empleats (empleats.component.ts)
    5. Creem la vista que representa al component
  • Creació del component detall de l'empleat
    1. La comanda ng g c components/empleat
    2. Recordem que hem d'actualitzar el fitxer de rutes
    3. Implementar el controlador d'empleat (empleat.component.ts)
    4. Creem la vista que representa al component
  • Creació d'un servei API REST
    1. Crear el servei per mostrar els posts. Recordem que aquesta información serà externa a la nostra aplicació.
    2. Hem d'importar en la secció providers el module HttpCLientModule (app.module.ts)
    3. Crearem el component que farà servir el servei
    4. Per fer la crida, fem servir el mètode subscribe.
    5. Crear el component per mostrar el detall dels posts
